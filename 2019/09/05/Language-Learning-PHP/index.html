函数

函数形参与实参

function 定义函数时，单括号中的为参数栏，它包括形参（可以以表达式的形式），默认值（缺省值），强类型参数，可变数量的参数列表，值传递和引用传递。例如，在调用上面的 foo( ) 函数时，单括号中的 1 是实参。如下：

function foo($arg_1, $arg_2, /*...*/, $arg_n){
    $sum = $arg_1 + ... + $arg_n;
    echo "Example function .\n";
    return $return_value;
}

foo(1..);

函数的默认值

function 定义函数时，若在参数栏给参数赋值，则在调用该函数时，如果缺省该参数，则使用该参数的缺省值。书写时，带缺省值的参数通常放到参数栏的最后，像下面的代码一样。

Example:

Code: 
function foo($arg_1, /*...*/, $arg_n, $arg_2=value){
    $sum = $arg_1 + ... + $arg_n;
    echo $arg_2;
    echo "Example function .\n";
    return $return_value;
}

foo(1, /*...*/, $arg_n); //lack $arg_2

Run:
value

函数强类型函数

一些简单类型的变量是可以互相转换的：

Example1:
Code:
$s = "123jkl";
$i = "12";
var_dump($s + $i);

Run: 
int(135) // 整型开头的字符串 + 整型数字 = 整型数字


Example2:
Code:
$s = "1.23jkl";
$i = "12";
var_dump($s + $i);

Run:
float(13.23) // 浮点型开头的字符串 + 数字 = 浮点型数字


Example3:
Code:
$s = "jk123";
$i = "jkd12";
var_dump($s + $i);

Run:
int(0) // 字符开头的串 + 字符开头的串 = 整型 0

强类型参数的定义：为参数列表中的参数指定类型。如果传入的数据类型不匹配，将会抛出 TypeError 异常。使用时需要在函数参数名前加上参数类型，并且同时需要限制类型转换，在程序开头声明： declare(strict_types=1)。这里要注意的是，strict_types 是在 PHP 7 版本下的。下面是各 PHP 版本的变量支持类型：

  • class/interface name PHP 5.0
  • array PHP 5.1
  • callable PHP5.4
  • bool/float/int/string PHP 7.0

Example：

Code:
declare(strict_types=1);
function sum_1($a, $b){ //正常传参
    return $a + $b;
}

function sum_2(int $a, int $b){ // 使用强类型
    return $a + $b;
}

echo sum_1(1, 1.5);
echo "\n";
echo sum_2(1, 1.5);

Run:
2
TypeError!

函数可变参数列表

可变参数列表与三个函数有关：

  • func_get_args( ): 结构说明：func_get_args ( void ) : array。在自定义函数内部使用，该函数返回一个包含函数参数列表的数组。
  • func_get_arg( ): 结构说明：func_get_arg ( int $arg_num ) : mixed。在自定义函数内部使用，该函数返回参数列表的某一项，其中 $arg_num 代表参数的列表中的第几项。
  • func_num_args( ): 结构说明：func_num_args( void ) : int。在自定义函数内部，该函数返回自定义函数中参数的数量。
  • 注意：以上三个函数通常出现于 PHP 5.5及以前*

Example

Code:
function get_num(){
    $args_num = func_num_args();
    $sum = 0;

    if(!$args_num){
        return $sum;
    }
    else{
        for($i=0;$i<$args_num;$i++){
            $sum += func_get_arg($i);
        }
        return $sum;
    }
}

echo get_num(), "<br>";
echo get_num(1,2,3), "\n";

Run:
0
6

在 PHP 5.6之后 ，我们使用另外一种方法：

Code:
function get_num2(...$nums){ //$nums作为参数列表数组
    $sum = 0;
    if(!$nums){
        return $nums;
    }
    else{
        foreach($nums as $num){
            $sum += $num;
        }
        return $sum;
    }
}

echo get_num2(), "<br>";
echo get_num2(1, 2, 3), "<br>";

Run:
Array //若不传参则返回一个数组
6

函数值传递与引用传递

函数正常传参不影响实参的值。使用引用则会影响实参的值，如下:

Code:
function swap(&$a, &$b){
    $tmp = $a;
    $a = $b;
    $b = $tmp;
}

$a1 = 3;
$b1 = 5;
swap($a1, $b1);
echo $a1, '<br>';
echo $b1, '<br>';

Run:
5
3

引用亦可以作为函数返回值的一种方式。引用传递的类型有：数值，对象，字符串，数组。

函数变量作用域

局部变量

几种情况：

  • 当前页面中声明的普通变量，不能在函数或者类中使用。
  • 当前页面中声明的普通变量，不能被传递到其他页面。
  • 在函数中声明的普通变量，仅在函数内部有效。
  • 在类中声明的属性，在类的内部有效。

全局变量

对于 PHP 而言，可以认为在页面中生成的变量对于该页面而言都是全局变量，在整个页面都是有效的，但是不能被函数或者类中的方法访问。若想要页面的变量被函数或者类中的方法访问，则需要使用超全局变量(gloabal或者GLOBALS[])。

Example:

Example1: //使用global对需要使用的函数外变量进行声明
Code:
$a = 1;
function test_global(){
    global $a;
    echo $a;
}

test_global();

Run:
1

Example2: //使用超全局变量GLOBAL数组
Code:
$a = 1;
function test_global(){
    echo $GLOBALS['a'];
}

test_global();

Run:
1

使用global声明变量和超全局变量GLOBALS数组的不同之处在于，global声明后的变量是依据指定的函数外的变量新创建的一个相同的变量，在函数内对这个变量操作不影响对应函数外变量的值。而GLOBALS数组中对应的变量就等同函数外的变量，对GLOBAL数组中的变量操作会影响函数外对应变量的值。
PHP 自带的超全局变量如下：

  • $_GET
  • _POST
  • _COOKIE
  • _SERVER
  • _FILES
  • _ENV
  • _REQUEST
  • _SESSION

静态变量（static）

函数内 static 声明的变量只在第一次有效。如下：

Example:

Code:
function test_static(){
    static $a = 0;
    echo $a++, "<br>";
}

test_static();
test_static();
test_static();
test_static();

Run:
0
1
2
3

注意 static 变量的赋值一定不能是函数表达式！
