类 与 对象

类用于确定对象将会拥有的特征（属性）和行为（方法）。

特点

  • 类是对象的类型。
  • 类是具有相同属性和方法的一组对象的集合。
  • 属性是对象的特定项。
  • 方法是对象执行的操作。
  • 类是抽象的概念，对象是一个确定的实体。
  • 所有的 JAVA 程序都以类（Class）为组织单元。

一个类可以映射多个具有相同属性和方法的对象，即一个类可以对应多个对象。

类的实现

定义类的步骤:

  • 定义类名（类名的首字母大写）
  • 编写类的属性
  • 编写类的方法（可以没有方法）

使用类的步骤

创建对象（实例化）

先了解下 JVM（JAVA 虚拟机）的内存空间，它主要由三部分组成：

  • 方法区内存（只有一处）
  • 堆内存（只有一处）
  • 栈内存（数量与线程有关，一个线程一个栈内存）

三块内存中变化最频繁的是栈内存，最先有数据的是方法区内存，垃圾回收主要针对的是堆内存。

创建语句：
类名 对象名 = new 类名( );
如：People joe = new People( );

创建过程如图：

  • 创建对象的过程中，程序会对电脑类进行实例化成为一个对象，在 JVM（JAVA 虚拟机） 堆内存当中开辟新的内存空间。
  • 实际上这里的 类名( ) 是构造方法。
  • 对象会涉及到 JAVA 虚拟机内的三个内存区域：方法区内存，栈内存和堆内存。
  • 方法区内存：在类加载的时候，类（Class）字节码代码片段会被加载分配到该内存空间中。
  • 栈内存（存放局部变量）：对象中方法代码片段执行的时候，会给方法分配内存空间，进行压栈操作。方法执行完成之后，其对应的内存空间释放，进行弹栈操作。 对象（在堆内存中）的首地址在 new 的时候赋值给对象名变量，对象名变量存放在栈内存中（内部类除外。内部类的引用为引用数据类型，可以作为成员变量）。由于对象名变量存储对象的首地址，我们称其为引用。而在堆内存中开辟对应的内存空间称为对象。
  • 堆内存：new 对象的成员变量的实例变量在堆内存中存储。

对象属性和方法的使用

引用对象的属性：joe.height = 5;
引用对象的方法：joe.walk();

成员变量和局部变量（MVs and LVs）

  • 成员变量：在类中定义，用来描述对象将要有什么。成员变量包括类变量（静态变量）和实例变量。类变量储存在类的静态域（在方法区内存中），实例变量储存在堆内存对应的对象内存中。
  • 局部变量：在类中的方法定义，在方法中临时保存数据，储存在栈内存中。

两者的区别：

  • 成员变量的作用域在整个类内部都是可见的；而局部变量的作用域仅仅限于定义它的方法。
  • JAVA 会给成员变量一个初始值（byte/short/long/int 类型为 0 ，float/double 类型为 0.0，boolean 类型为 false，char 类型为 \u0000， 引用数据类型为 NULL）；而不会给局部变量赋予初始值。
  • 在方法内部，出现方法内局部变量和类的成员变量同名时，局部变量具有更高的优先级。
  • 成员变量的实例变量储存在堆内存中；局部变量储存在栈内存中。

构造方法

构造方法属于方法，是定义在 JAVA 类中的一个用来初始化对象的方法。它与类同名并且没有返回值。

特点：

  • 如果定义类的时候没有写入构造方法，编译时JAVA 会自动补充构造方法。
  • 如果定义类的时候写入了构造方法，则无论是有参、无参的构造方法，都不会自动补充构造方法。
  • 有参的构造方法可以和无参的构造方法并存。（参考函数重载）
  • 构造方法给属性赋值，并且可以设置值的条件。

静态变量（Static）

静态变量属于成员变量。它存储在类的静态域（静态域在方法区内存中）。

垃圾回收机制（GC 机制）

对于一个堆内存的对象，如果后继代码中没有更多的对象指向它或者它无法被访问的时候，这个对象会变为垃圾数据，按照GC 机制，垃圾回收器会将垃圾进行回收。

空指针异常（null pointer exception）

空引用数据类型（null）访问对应类实例相关（区别于静态）的数据（包括实例变量和实例方法）会引发空指针异常。

对于实例变量：
Example:
Class apple

public class apple {
    String color = "red";

    peer p;

}

Main Function

public static void main(String[] args) {

    apple a1 = new apple();
    a1 = null;
    System.out.println(a1.color);
}

Result

Exception in thread "main" java.lang.NullPointerException
    at test.main(test.java:15)

对于实例方法：
Example:
Class apple

public class apple {
    String color = "red";

    peer p;    

    public void price() {  // 实例方法
        System.out.println("1.23");
    }

}

Main Function

public static void main(String[] args) {
    apple a1 = new apple();

    a1 = null;

    a1.price();
}

Result

Exception in thread "main" java.lang.NullPointerException
    at test.main(test.java:14)
