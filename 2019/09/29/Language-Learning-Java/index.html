对象（Object）

类（Class）

类用于确定对象将会拥有的特征（属性）和行为（方法）。

特点

  • 类是对象的类型。
  • 类是具有相同属性和方法的一组对象的集合。
  • 属性是对象的特定项。
  • 方法是对象执行的操作。
  • 类是抽象的概念，对象是一个确定的实体。
  • 所有的 JAVA 程序都以类（Class）为组织单元。

可以用一个类表示多个具有相同属性和方法的对象，即一个类可以对应多个对象。

类的实现

定义类的步骤:

  • 定义类名
  • 编写类的属性
  • 编写类的方法（可以没有方法）

使用类的步骤：

  • （一）创建对象（实例化）：
    类名 对象名 = new 类名( );
    如：People joe = new People( );
    创建对象后，程序会对电脑类进行实例化成为一个对象。
    如图：
  • （二）使用对象：
    引用对象的属性：joe.height = 5;
    引用对象的方法：joe.walk();

成员变量和局部变量

  • 成员变量：在类中定义，用来描述对象将要有什么。一些成员变量可以在其他类中被使用。

  • 局部变量：在类中的方法定义，在方法中临时保存数据。

两者的区别：成员变量的作用域在整个类内部都是可见的；而局部变量的作用域仅仅限于定义它的方法。
