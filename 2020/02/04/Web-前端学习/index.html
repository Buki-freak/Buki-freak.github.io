document.getElementById(elementID : String)
An ID should be unique within a page. However, if more than one element with the specified ID exists, the getElementById() method returns the first element in the source code.

element.classList.toggle(class, true|true)
The first parameter removes the specified class from an element, and returns false. *
*If the class does not exist, it is added to the element, and the return value is true.

Timing 事件

Definiton
The window object allows execution of code at specified time intervals.These time intervals are called timing events.

Example:

使用 setTimeout。

注意：

window.setTimeout(function, milliseconds : number)
Executes a function, after waiting a specified number of milliseconds.

使用 setInterval。

注意：
window.setInterval(function, milliseconds : number, param1, param2, …)
Same as setTimeout(), but repeats the execution of the function continuously.
param1, param2, … Optional. Additional parameters to pass to the function.
