通过 window.document 修改 标签 的内容。

<!DOCTYPE html>

目录的开合范例

Example:

<!-- Feb.4th, 2020 -->
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8">
    <title>HTML DOM 选单开合操作</title>
    <script type="text/javascript">
    function toggleMenu(number) {
        var menu = document.getElementById("menu_"+number);
        menu.classList.toggle("hide"); // 类名切换
    }
    </script>
    <style type="text/css">
    .hide {
        display: none;
    }
    </style>
</head>

<body>
    <div onclick="toggleMenu(1);">选单的标题 1</div>
    <ul id="menu_1">
        <li>项目 1-1</li>
        <li>项目 1-2</li>
        <li>项目 1-3</li>
    </ul>
    <div onclick="toggleMenu(2);">选单的标题 2</div>
    <ul id="menu_2">
        <li>项目 2-1</li>
        <li>项目 2-2</li>
        <li>项目 2-3</li>
    </ul>
    <div onclick="toggleMenu(3);">选单的标题 3</div>
    <ul id="menu_3">
        <li>项目 3-1</li>
        <li>项目 3-2</li>
        <li>项目 3-3</li>
    </ul>
</body>

</html>

注意：

document.getElementById(elementID : String)
An ID should be unique within a page. However, if more than one element with the specified ID exists, the getElementById() method returns the first element in the source code.

element.classList.toggle(class, true|true)
The first parameter removes the specified class from an element, and returns false. *
*If the class does not exist, it is added to the element, and the return value is true.

Javascript

事件处理

Example:

使用 addEventListener。

<!-- Feb.2nd, 2020 -->
<!-- Javascript 事件动态处理-->
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8">
    <title>Event 事件处理</title>
    <script type="text/javascript">
    // 用 Javascript 的程式，动态的注册事件处理器

    function init() { // 进行动态事件处理器注册
        /* 这个为旧版本的写法
        var btn = document.getElementById("btn");
        btn.onclick = function() { // 该函数称为事件处理器，on + 事件名称
            alert("Click");
        };
        */


        /* 新版本的写法
        var btn = document.getElementById("btn");
        function handler(){ // 准备一个事件处理器
            alert("Click");
        };
        btn.addEventListener("click", handler); // click 为事件名称
        */
    }
    </script>
</head>

<body onload="init();">
    <!-- 动态注册，通过 load 事件进行，在 HTML 载入完成后进行 -->
    <button id="btn" onclick="alert('in');">Click</button> <!-- 静态注册，写在标签上，在载入 HTML 的时候进行 -->
</body>

</html>

注意：

使用 addEventListener 动态添加的处理器不会将静态处理器覆盖。

事件对象 EventObj

事件对象一般用 e 表示。

Example:

<!-- Feb.4th, 2020 -->
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8">
    <title>事件物件 Event Object</title>
    <script type="text/javascript">
    function init() {
        var btn=document.getElementById("btn");
        var handler=function(e){ // 事件处理器的第一个参数默认为事件对象 e
            alert(e.clientX+","+e.clientY);
        }
        btn.addEventListener("click", handler);
        /*
            1. 使用者点击了按钮，触发 click 事件。
            2. 浏览器主动收集和事件有关的资讯，，并产生 Event Object 事件物件。
            3. 呼叫已经注册的事件处理器（事件处理函数）
                handler(eventObj);
         */

        /*
        document.addEventListener("keydown", function(e){
            alert(e.keyCode); // 获得按键的 ASCII 码
        });
        */
    }
    </script>
</head>

<body onload="init();">
    <button id="btn">Click</button>
</body>

</html>

注意：

element.addEventListener(event : String, functionname : function)
Do not use the “on” prefix. For example, use “click” instead of “onclick”.

Timing 事件

Definiton
The window object allows execution of code at specified time intervals.These time intervals are called timing events.

Example:

使用 setTimeout。

<!-- Feb.4th, 2020 -->
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8">
    <title>自动排程 Schedule</title>
    <script type="text/javascript">
    function init() {
        window.setTimeout(function(){
            alert("Hello world");
        }, 2000);
    }
    </script>
</head>

<body onload="init();">
</body>

</html>

注意：

window.setTimeout(function, milliseconds : number)
Executes a function, after waiting a specified number of milliseconds.

使用 setInterval。

<!-- Feb.4th, 2020 -->
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8">
    <title>自动排程 Schedule</title>
    <script type="text/javascript">
    function init() {
        window.setInterval(function(){
            alert("Hello world");
        }, 2000);
    }
    </script>
</head>

<body onload="init();">
</body>

</html>

注意：
window.setInterval(function, milliseconds : number, param1, param2, …)
Same as setTimeout(), but repeats the execution of the function continuously.
param1, param2, … Optional. Additional parameters to pass to the function.

CSS

线性转换

包括 位移，缩放，旋转，斜歪。

Example:

四种变换。

<!-- Feb.5th, 2020 -->
<!DOCTYPE html>
<html>

<head>
    <meta charset="utf-8">
    <title>CSS 3 线性转换： 位移、缩放、旋转、斜歪</title>
    <style type="text/css">
    .box {
        width: 100px;
        height: 100px;
        background-color: red;
        transform: translate(100px, 50px); /* 位移。 param1 代表 X 方向的位移; param2 代表 Y 方向的位移 */
        transform: scale(1.5, 1); /*缩放。 param1 代表 X 方向缩放倍数; param2 代表 Y 方向缩放倍数*/
        transform: rotate(30deg); /*旋转。 param 代表顺时针旋转的度数。*/
        transform: skew(45deg, 0deg); /*斜歪。 param1 代表 X 方向的斜歪; param2 代表 Y 方向的斜歪*/

        transform: translate(100px, 50px) scale(1.5, 1) rotate(30deg) skew(45deg, 0deg); /* 设置需要综合在一起写 */
    }
    </style>
</head>

<body style="margin: 0px;">
    <div class="box"></div>
</body>

</html>
